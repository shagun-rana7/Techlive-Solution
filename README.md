# Techlive Solution

Techlive Solution is a full-stack web project developed during my MERN Stack internship.

## Features
- Admin Panel to manage website data
- Admin can add, update, and delete data
- Users can view data fetched from the backend
- REST API integration

## Technologies Used
- React.js
- Node.js
- Express.js
- MongoDB
- HTML, CSS, Bootstrap, JavaScript
